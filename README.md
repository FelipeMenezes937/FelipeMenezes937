## Projetos

### HeapSec Antivirus
> Scanner antivírus 100% local por análise estática, com múltiplos motores de detecção heurística. Desenvolvido em Java puro, sem dependências externas, processando ~1.800 arquivos/s com ForkJoinPool.

**Tech Stack:**
![Java](https://img.shields.io/badge/Java-ED8B00?style=flat-square&logo=openjdk&logoColor=white)
![Shell](https://img.shields.io/badge/Shell-4EAA25?style=flat-square&logo=gnubash&logoColor=white)
![YARA](https://img.shields.io/badge/YARA-443203?style=flat-square&logo=lock&logoColor=white)

**Diferenciais técnicos:**
- Múltiplos motores: Entropia Shannon, pattern matching Aho-Corasick, parsing PE nativo
- 60+ padrões comportamentais (password stealers, RATs, cryptominers, bankers)
- Proteção contra path traversal, symlink attacks e zip bombs
- Zero dependências — funciona offline

[![Check Project](https://img.shields.io/badge/GitHub-Check%20Project-181717?style=for-the-badge&logo=github)](https://github.com/FelipeMenezes937/heapsec)
